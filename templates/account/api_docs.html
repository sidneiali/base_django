Headers

Headers importantes

Quase todos os endpoints exigem Bearer token. Em toda resposta da API, use o X-Request-ID para rastrear chamadas e auditoria. 

Authorization: Bearer SEU_TOKEN
Accept: application/json
X-Request-ID: 6472beea-7623-43fa-aa4c-34b117d74b99
Content-Type: application/json

Teste local

Como testar pelo script

Preencha scripts/.env com seu token local e use o utilitário versionado em scripts/api_request.py para ver status, headers e body formatado. 

python scripts/api_request.py --help
python scripts/api_request.py --list-routes
python scripts/api_request.py /api/v1/core/me/
python scripts/api_request.py /api/v1/core/audit-logs/
