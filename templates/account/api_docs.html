Swagger da API

Página pública para explorar e testar a API sem carregar o shell com sidebar. O acesso aos endpoints segue protegido por Authorization: Bearer <token>.

Recursos

Usuários do painel

Base URL: /api/panel/users/
Gere seu token em Minha conta e envie no header Bearer.
  • GET `/api/panel/users/`
  • POST `/api/panel/users/`
  • GET `/api/panel/users/<id>/`
  • PATCH `/api/panel/users/<id>/`
  • DELETE `/api/panel/users/<id>/`

Logs de auditoria

Base URL: /api/core/audit-logs/
  • GET `/api/core/audit-logs/`
  • GET `/api/core/audit-logs/<id>/`
Exemplos rápidos

Chamadas com Bearer token

Estes exemplos cobrem os recursos de usuários do painel e logs de auditoria.
Baixar Postman JSON
  • curl
  • Python

GET `/api/panel/users/`

curl -H "Authorization: Bearer SEU_TOKEN" {{ api_base_url }}/api/panel/users/

POST `/api/panel/users/`

curl -X POST {{ api_base_url }}/api/panel/users/ \
  -H "Authorization: Bearer SEU_TOKEN" \
  -H "Content-Type: application/json" \
  -d "{\"username\":\"api-user\",\"email\":\"api@example.com\",\"password\":\"SenhaSegura@123\",\"is_active\":true}"

GET `/api/panel/users/<id>/`

curl -H "Authorization: Bearer SEU_TOKEN" {{ api_base_url }}/api/panel/users/1/

PATCH `/api/panel/users/<id>/`

curl -X PATCH {{ api_base_url }}/api/panel/users/1/ \
  -H "Authorization: Bearer SEU_TOKEN" \
  -H "Content-Type: application/json" \
  -d "{\"email\":\"alterado@example.com\"}"

DELETE `/api/panel/users/<id>/`

curl -X DELETE {{ api_base_url }}/api/panel/users/1/ \
  -H "Authorization: Bearer SEU_TOKEN"

GET `/api/core/audit-logs/`

curl -H "Authorization: Bearer SEU_TOKEN" {{ api_base_url }}/api/core/audit-logs/

GET `/api/core/audit-logs/<id>/`

curl -H "Authorization: Bearer SEU_TOKEN" {{ api_base_url }}/api/core/audit-logs/1/

GET `/api/panel/users/`

import requests

response = requests.get(
    "{{ api_base_url }}/api/panel/users/",
    headers={"Authorization": "Bearer SEU_TOKEN"},
    timeout=30,
)

print(response.status_code)
print(response.json())

POST `/api/panel/users/`

import requests

payload = {
    "username": "api-user",
    "email": "api@example.com",
    "password": "SenhaSegura@123",
    "is_active": True,
}

response = requests.post(
    "{{ api_base_url }}/api/panel/users/",
    headers={"Authorization": "Bearer SEU_TOKEN"},
    json=payload,
    timeout=30,
)

print(response.status_code)
print(response.json())
